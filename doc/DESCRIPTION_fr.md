Open Web Analytics est une alternative open source aux outils d'analyse Web commerciaux tels que Google Analytics. Ce logiciel vous permet de garder le contrôle des données que vous collectez sur les utilisateurs de vos sites Web ou applications.

### Fonctionnalités

    Suivi des visiteurs, des pages vues, des transactions e-commerce et des actions configurables
    Suivez un nombre illimité de sites Web à l'aide d'une seule instance du serveur OWA
    Client de suivi Javascript de première partie
    Tableau de bord/portail de rapports
    Affichez et personnalisez tous les rapports
    Générez des cartes thermiques
    Générez des enregistrements de session « Domstream »
    Géolocalisation des visiteurs
    API REST pour l'administration et l'accès aux données
    Interface de rapports multi-utilisateurs
    Cadre extensible via des modules personnalisés
